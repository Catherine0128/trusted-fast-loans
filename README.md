\# Trusted Fast Loans System

A web-based loan management system built using Python Flask and SQLite.



\## Features

* Loan application form
* Admin login authentication
* Admin dashboard
* Approve/Reject loan request
* Loan status tracking



\## Technologies Used

* Python(Flask)
* HTML \& CSS
* SQLite Database



\## How to Run

1. Install dependencies

&#x20; pip install-r requirements.txt



2\. Run the app

&#x20; python app.py



3\. Open in browser

&#x20; http://127.0.0.1:5000



\## Project Description 

&#x20;This system allows users to apply for loans and administrators to manage and approve applications through a dashboard.
